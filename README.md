# OCR을 이용한 책장 관리 및 보유 도서 기반 책 추천 서비스

# 📋프로젝트 배경 및 목적
  한강 작가의 노벨문학상 수상을 전후로 독서에 대한 관심 및 도서 구매량이 증가하고 있다. 이에 반해 보유한 도서를 관리하는 서비스는 부족한 실정이다.
이에 보유 도서를 기반으로 하는 개인화된 책 추천 시스템을 구현하여, 모바일에서 편리하게 이용 가능한 웹 서비스를 개발한다.
